# Remotion設計書 v2

## A. 参照動画の要約

この動画は、固定背景 + 固定レイアウト + 固定タイミング で構成された、かなり規則性の強いランキング動画です。

- 構成は Intro 1本 + Ranking 10シーン + Outro 1本
- 各ランキングシーンは 完全に同じ尺・同じレイアウト・同じ表示順
- シーン切替は 1フレーム黒フラッシュ → 新シーン暗状態から約10フレームで明転
- 背景は オレンジ/黄の放射状サンバースト で、静止。パン/ズーム/回転は見えない
- 各ランキングシーンは2段構成
  - 順位 + 中央の2行見出し + 下部中央の正方形画像A
  - 順位 + 上段コメントボックス + 下段コメントボックス + 下部中央の正方形画像B
- 実際の参照動画では画像未設定のため、画像は Fallback 01 / Fallback 02 のプレースホルダで表示されている
  → つまりテンプレート上は 各シーンにつき画像スロットが2つ ある前提で設計するのが自然
