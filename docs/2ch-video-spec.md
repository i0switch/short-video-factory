# 2ch風ショート動画 生成仕様書

参考動画5本の解析結果に基づく、2ch風ショート動画の生成ルール。
台本生成・レンダリングの両方でこの仕様に従うこと。

## 1. 画面レイアウト（1080x1920）

### 安全マージン
- 左右: 40-60px (約4-6%)
- 上下: 20-40px

### 絶対ルール: 字幕とキャラクターは被らない
- 字幕とキャラの間に最低15-20%の空間を確保
- キャラのスケールを上げすぎない（max 0.85、closeupでも1.0まで）

## 2. タイトルバンド

- **位置**: 画面最上部、常時表示
- **背景**: 純黒 (#000000)、半透明でもOK
- **行数**: 2行
  - 1行目: シリーズ名（例: "笑える迷語集"）
  - 2行目: エピソード名（例: "彼女の手料理がヤバすぎた話"）
- **フォント**: 極太ゴシック (Noto Sans JP Black 相当), 80-120px
- **色**: 黄橙 (#FFD700〜#FFA500) + 黒ストローク (5-8px)
- **zIndex**: 最上位 (100)
- **レイヤー順**: 常に他のすべての要素の上

## 3. 字幕（キャプション）

### 基本仕様
- **位置**: y: 12-35% (タイトルバンドの直下)
- **配置**: 左寄せ（短いテキストは中央でもOK）
- **フォント**: 極太ゴシック, 70-100px
- **行数**: 1-3行, 1行5-9文字
- **ストローク**: 黒 3-8px + ドロップシャドウ
- **zIndex**: 60

### 話者別カラールール
| 話者 | 色 | 用途 |
|---|---|---|
| narrator (ナレーター) | ライトブルー (#4FC3F7) | 地の文、状況説明 |
| character1 (主人公/男) | 白 (#FFFFFF) or 黄 (#FFEB3B) | メイン話者 |
| character2 (ツッコミ) | 赤 (#F44336) | ツッコミ、リアクション |
| character3 (女性) | ピンク (#FF69B4) or 黄 (#FFEB3B) | 女性キャラ |
| 威圧系キャラ | オレンジ (#FF9800) | ボス、姑など |

### 表示ルール
- 字幕はTTS音声と同時に出現
- ハードカット（フェードなし）で切り替え
- 表示時間: 通常1-3秒、オチ2-3.5秒

## 4. キャラクター

### 素材
- **スタイル**: いらすとや風フラット水彩イラスト
- **形式**: 背景透過PNG
- **サイズ**: 画面幅の50-80%

### 配置ルール
| パターン | 配置 |
|---|---|
| 1人 | 中央、画面幅60-80% |
| 2人 | 左右分割、メイン大きく |
| 3人 | 三角配置 or 横一列 |

### 位置 (anchor: bottom-center)
- **ソロ**: x:50%, y:85-95% (画面下部)
- **左キャラ**: x:30%, y:85-95%
- **右キャラ**: x:70%, y:85-95%
- **scale**: 0.6-0.85（closeup時のみ1.0まで）

### アニメーション
- **微揺れ**: 常時 +/-2-5px（呼吸感）
- **ズームイン**: scale 1.0→1.2-1.5, 0.3s, ease-out（感情強調時）
- **スライドイン**: 画面外から0.3sで登場
- **表情変化**: 画像差し替え（1フレーム、アニメーションなし）

## 5. 背景

### 種類
- **場所背景**: 3D風/イラスト風（居酒屋、キッチン、廊下、寝室、オフィス等）
- **感情背景**: 感情ピーク時に場所背景を置換

### 感情別背景パレット
| 感情 | 背景色/パターン | 使用タイミング |
|---|---|---|
| neutral | ベージュ/茶色 | 日常シーン |
| happy/punchline | 黄金ラジアル (#FFD700) | オチ、祝い |
| anger | 赤 + 集中線 | 怒り爆発 |
| confusion | 紫渦巻き | 困惑、混乱 |
| shock | 黒 + 稲妻 | 衝撃シーン |
| sad | 暗青 + 雨線 | 失望、悲しみ |
| positive | 黄 + ボケ円 | 前向き評価 |

## 6. エフェクト

### 使用タイミングと種類
| エフェクト | 見た目 | タイミング |
|---|---|---|
| concentration_lines | 白放射線 | サプライズ、衝撃 |
| confetti | 紙吹雪 | オチ、お祝い |
| vortex | 紫渦巻き背景 | 困惑 |
| lightning | 白稲妻（黒背景） | 衝撃発覚 |
| rain | 青縦線 | 失望、ガッカリ |
| sparkle | キラキラ星 | ポジティブ |
| shake | 画面揺れ | 怒り、衝撃 |
| flash | 白フラッシュ | インパクト |

### マンガ記号
- **!?マーク**: 赤、キャラ頭上、バウンスアニメ (0→1.2→1.0)
- **汗マーク**: 白しずく、キャラ頭付近
- **怒りマーク**: 赤、キャラ頭上
- **///（照れ）**: ピンク斜線
- **💢（怒り）**: 赤い記号

## 7. シーン構成パターン

### Beat構造（物語上の役割）
| Beat | 位置 | 典型的な演出 |
|---|---|---|
| intro | 最初 | タイトル表示、場面説明 |
| setup | 序盤15% | 日常シーン、neutral背景 |
| incident | 中盤前 | 事件発生、背景変化開始 |
| reaction | 中盤 | リアクション、エフェクト |
| escalation | 後半 | 状況悪化、anger/confusion |
| climax | クライマックス | 最大エフェクト、ズームイン |
| aftermath | 終盤 | 収束、引きの構図 |
| outro | 最後 | チャンネル登録促し |

### シーンタイミング
- 通常シーン: 1-3秒
- オチシーン: 2-3.5秒（長めに見せる）
- 衝撃リアクション: 0.5-1.5秒（短く鋭く）
- 余韻シーン（字幕なし）: 2-3秒

### トランジション
- **99%ハードカット**（フェード・スライドなし）
- エピソード間のみモーションブラー(0.2s)の場合あり

## 8. 音声

### TTS
- ナレーター: ずんだもん
- キャラ1（男）: 青山龍星
- キャラ2（女）: 四国めたん
- 読み速度: 5-7文字/秒

### BGM
- コメディ/チップチューン系
- 常時再生、途切れない
- 音量: -20dB程度（ナレーションより小さく）

### SE
- エフェクトと同時に発火
- 集中線/稲妻/フラッシュ時に効果音

## 9. レイヤー順序（下から上）

1. 場所背景 (zIndex: 0)
2. 感情背景/エフェクト背景 (zIndex: 5)
3. サブキャラクター (zIndex: 10)
4. メインキャラクター (zIndex: 20)
5. 小道具/プロップ (zIndex: 25)
6. マンガ記号 (zIndex: 30)
7. 字幕テロップ (zIndex: 60)
8. タイトルバンド (zIndex: 100)
9. 全画面エフェクト (zIndex: 90)

## 10. 台本生成時の素材指定ルール

### 必須フィールド
```json
{
  "speaker": "narrator",
  "text": "セリフ（20文字以内）",
  "emotion": "neutral",
  "effect": "none",
  "characterSrc": "assets/nicocommons/char_male_suit.png",
  "backgroundSrc": "assets/nicocommons/bg_kitchen.png",
  "imageKeywords": ["日本語KW"],
  "imageKeywordsEn": ["english KW"]
}
```

### 素材選定優先順位
1. ニコニ・コモンズ取得済み素材 → characterSrc/backgroundSrc直接指定
2. いらすとや → imageKeywordsで自動検索
3. Pexels → imageKeywordsEnでフォールバック

| bg_hallway.png | 廊下背景 |
