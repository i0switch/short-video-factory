# 動画再現プロンプト — 完全仕様書

## ■ 目的
添付の「最終理想成果物.mp4」を完全に再現するプログラム（Remotion / FFmpeg / Canvas等）を実装するための仕様書。
文字の動き・タイミング・背景アニメーション・レイアウト・色・すべてを一致させること。
テキスト内容、背景画像、キャラクター素材は差し替え可能な変数として設計すること。

## ■ グローバル要素

### 背景：回転サンバースト（全シーン共通・常時アニメーション）
- **タイプ**: 放射状の光線（サンバースト / radial rays）
- **中心点**: キャンバス中央（540, 960）
- **光線の本数**: 約36本（18ペア × 明暗交互）
- **色**:
  - 明るい光線: `#F5B731`（ゴールデンイエロー）
  - 暗い光線: `#D4871A`（ダークオレンジ）
- **回転**: 時計回り、一定速度、1回転あたり約20-24秒（約15°/秒）
- **回転は全シーンを通して途切れず連続**する。シーン遷移でリセットしない。
- **実装ヒント**: `rotation = (frame / fps) * 15` 度で回転し、`ctx.translate(centerX, centerY) → ctx.rotate(rad) → draw triangles` のパターン

## ■ シーン遷移

- **全シーン間の遷移はハードカット（即切り替え）**。フェードやディゾルブは使わない。
- サブシーンA→B→C内では要素の追加表示（カット切り替えではなく、同一画面上に要素が追加される）

---

## ■ テキスト描画の共通ルール（縁取り文字の実装）

すべてのテキストは以下の手順で描画：
1. `ctx.strokeStyle = "#000000"`, `ctx.lineWidth = 縁取り幅`, `ctx.lineJoin = "round"` でstrokeText
2. `ctx.fillStyle = 指定色` でfillText
3. 影: `ctx.shadowColor = "rgba(0,0,0,0.5)"`, `ctx.shadowOffsetX = 3`, `ctx.shadowOffsetY = 3`, `ctx.shadowBlur = 4`
4. **strokeを先、fillを後** にすることで、縁取りが文字の外側にのみ出る

## ■ 実装時の注意

1. **サンバースト回転は全シーン通して連続**。シーンが切り替わっても角度はリセットしない。
2. **テキストの縁取りは必ず stroke → fill の順**。逆にすると縁取りが文字を覆う。
3. **コメントボックスは2種類**あり、見た目が異なる（青枠白背景 vs 赤背景白文字）。
4. **キャラ画像はランクごとに差し替え**。同じキャラが複数ランクで使われるケースもある。
5. **自動改行ロジック**: テキスト幅がボックス幅を超える場合、自動で改行する処理が必要。
6. **全テキスト・画像パスは外部JSONから読み込み**、ハードコードしない。
7. シーン間のタイミングは上記JSONの `duration_sec` で制御し、合計が目標尺になるよう調整。
